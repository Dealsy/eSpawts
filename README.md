
eSpawts is currently a work in progress, this is a game tournament site that allows users to create tournaments for their favorite games,
and then assign a monetized prize pool to their tournament.

## Program Stack

### Frontend
 -  React.JS
 -  React Router
 -  Tailwind.CSS
### Backend
 -  ASP.Net
